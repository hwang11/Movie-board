# 영화 리뷰 작성 게시판

## 💡기능

### 리뷰 게시판 기능

- 게시글 조회
- 게시글 등록
- 게시글 수정
- 게시글 삭제

### 회원 기능

- 구글/네이버 로그인
- 로그인한 사용자 글 작성 권한
- 본인 작성 글에 대한 권한 관리

### 영화 검색 기능 ( 네이버 API )

- 영화 검색
- 블로그 검색

## 📚기술

SpringBoot 2

Spring Data JPA

Spring Security

Mustache

AWS 

- EC2
- RDS
